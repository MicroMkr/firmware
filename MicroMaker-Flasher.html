First-Time Setup

  • 01 After flashing, press RESET on the board if it doesn't reboot automatically.
  • 02 The radio will create a WiFi access point called MyBT_Radio — connect to it.
  • 03 Wait ~2 minutes for the flash to format on first boot. Restart the radio; the LCD will show the local IP.
  • 04 Open that IP in your browser to access the Web UI and upload your station list.
  • 05 Station list format: Station Name,http://stream.url — one per line.
  • 06 The Product ID can be found at the bottom of the webpage. Send that number to themicromaker@outlook.com.
  • 07 You can purchase a license key by sending $20 to tmicromaker@gmail.com PayPal account or through Tindie Store (https://www.tindie.com/stores/6659291/)
  • 08 Right after purchasing a license key, it will be sent to you via return email.
If flashing fails, hold BOOT, press & release RESET, then release BOOT to manually enter bootloader mode. Try a lower baud rate (460800) if you see write errors.
